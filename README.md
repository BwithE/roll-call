# Download
```
git clone https://github.com/bwithe/roll-call

cd roll-call

```

# Usage
```
chmod +x roll-call.sh

./roll-call.sh -t <IP>
# OR
bash roll-call.sh -t <IP>

```

```
./roll-call.sh -t 127.0.0.1
# OR
bash roll-call.sh -t 127.0.0.1

```
